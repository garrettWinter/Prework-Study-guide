HTML

  • A page has 3 main parts, Head, body, footer
  • h1 should only be used once on a page
  • The head doesnt include any details the user will see with the exception of the title and contains meta data about the page
  • The terms tag and elements are sometimes used interechangable, but are different.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Inline CSS - This applies to specific HTML tags using a style attributes for specific element.
  • Internal CSS Style Sheet - The details of the CSS are included directly to the head part of the HTML.
  • External CSS Style Sheet - A link to the CSS file is made in the header
